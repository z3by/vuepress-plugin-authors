# vuepress-plugin-editors

> All authors of a vuepress page

adds all authors of a file to the $page object

## Installation

```bash
yarn add -D vuepress-plugin-authors
// or npm install vuepress-plugin-authors -D

```

## Usage

```js
// .vuepress/config.js

module.exports = {
    plugins: [
      ['authors'],
      // other plugins
    ]
}
```
